# Appetiser Coding Exam

Todo app created using laravel and VueJS.

## Dependencies
- PHP 8
- vue
- Mysql
- Tailwind CSS
- Flowbite
- v-viewer

## Build Setup

```bash
# install php dependencies
$ composer install

# install vue dependencies

# copy environment variable examples
$ cp .env.example .env

# serve with hot reload at localhost:3000
$ php artisan serve

# serve vue app

$ npm run dev

```
